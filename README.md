Buddhist Art Heritage

A web platform connecting traditional Buddhist artists with art lovers worldwide. Enables artists to showcase their sacred artwork and receive custom commission requests securely.

🎯 Project Overview

Buddhist Art Heritage is a complete custom artwork marketplace where:
- Artists can create profiles, upload artwork, and manage custom requests
- Clientscan browse gallery, request custom pieces, and communicate with artists
- Admin manages users, artists, and artwork approval

✨ Features

👨‍🎨 For Artists
- Profile management with biography and specialization
- Artwork upload with descriptions, materials, and symbolism
- Manage custom artwork requests
- Real-time messaging with clients
- Track earnings and request status

👤 For Clients
- Browse gallery with deity filtering and sorting
- Like and comment on artworks
- Request custom artwork with detailed specifications
- 50% advance + 50% remaining payment via Khalti
- Real-time chat with artists
- Download purchased artwork

🔧 For Admin
- Dashboard with statistics
- Manage users, artists, and artworks
- Approve artist registrations
- Monitor custom requests

🧠 Algorithm

The gallery uses a popularity-based recommendation algorithm:

Formula:`Popularity Score = (Views × 0.3) + (Likes × 0.5) + (Comments × 0.2)`

- Views have 30% weight
- Likes have 50% weight (highest importance)
- Comments have 20% weight

Artworks with higher scores appear first in the gallery.

💰 Payment Flow

Integrated with Khalti payment gateway (test mode with demo backup).
